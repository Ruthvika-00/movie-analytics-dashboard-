# 🎬 Movie Analytics Dashboard

## 📌 Overview
This project analyzes movie data to uncover trends in ratings, popularity, and genre distribution using Tableau.

## 📊 Dashboard Features
- Top 10 highest-rated movies
- Genre distribution analysis
- Ratings vs Popularity scatter plot
- Movies released over time (trend analysis)

## 🧠 Key Insights
- High popularity does not always correlate with high ratings
- Action and Drama genres dominate movie production
- Movie releases have increased significantly over time

## 🛠️ Tools Used
- Tableau
- Google Sheets / CSV
